My Projects

Showcase of My Data Science Projects

*****

Sarcasm Sniffer – Sarcasm Detection Web Application

Developed a web application utilizing Flask, along with LSTM and Logistic Regression models for sarcasm detection. Achieved accuracies of 75.4% and 89.3% respectively. The application leverages real-time Twitter data and Kaggle datasets, integrated using Google Colab.

*****

Resume Screening App using NLP and Streamlit

Created an innovative resume screening application using Python and Streamlit, employing KNN algorithms for effective screening. The project was successfully deployed in Jupyter Notebook and PyCharm.

*****

Password Door-Lock System (IoT - Arduino Project)

Developed an IoT-based Password Door-Lock System using Arduino Tinkercad and the C Programming language. The project included essential components such as an Arduino Board, Servo motor, LED, Buzzer, PIR Sensor, Resistor, LCD display, and keypad.

*****

Grocery Store Android Application

Developed a Grocery Store Android Application using Android Studio and Java. This user-friendly application allows customers to purchase and order groceries online, featuring an attractive graphical user interface (GUI).
